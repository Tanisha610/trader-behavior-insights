# Trader Behavior Insights – Bitcoin Market Sentiment Analysis

## Objective
Analyze the relationship between trader performance and Bitcoin market sentiment
(Fear vs Greed) using Hyperliquid historical trade data.

## Datasets Used
- Bitcoin Fear–Greed Index
- Hyperliquid Historical Trader Data

## Methodology
- Aligned intraday trade data with daily market sentiment
- Normalized sentiment into Fear and Greed regimes
- Analyzed Closed PnL, trade size, and win rate
- Visualized performance distributions using boxplots

## Key Insights
- Traders exhibit higher risk-taking behavior during Greed phases.
- Greed periods show higher volatility and larger profit/loss dispersion.
- Fear periods reflect more conservative trading behavior.
- Neutral sentiment corresponds to minimal exposure.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
